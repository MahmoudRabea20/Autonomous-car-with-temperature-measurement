# Project Description 
<p>Autonomous car by microcontroller STM32F401CC with Ultrasonic sensor, and temperature sensor to measure the air temperature. </p> 
# Used Drivers 
<br>-RCC 
<br>-GPIO 
<br>-NVIC 
<br>-SYSTICK 
<br>-Timer2 
<br>-LCD 
<br>-ADC
